# Learn_LVGL

## 介绍

这是一个学习LVGL的仓库, 由于暂时没有开发板, 所以只能在PC上模拟运行, 所以克隆的仓库是[lv_port_pc_visual_studio](https://github.com/lvgl/lv_port_pc_visual_studio), 该仓库是LVGL官方提供的一个在PC上模拟运行的例程, 由Visual Studio 2022编译, 本仓库是在该仓库的基础上Fork的, 并在此基础上进行学习.
